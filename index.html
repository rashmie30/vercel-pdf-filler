PDF Filler API

This API allows you to fill PDF forms with data and make the fields read-only (flattened).

API Endpoints

GET /health

Check if the API is running.

POST /fill-pdf

Fill a PDF form and get a JSON response with the base64-encoded filled PDF.

POST /fill-pdf-binary

Fill a PDF form and get the binary PDF file as a response.

Example Request Format

[
  {
    "fileUrl": "https://example.com/path/to/template.pdf"
  },
  {
    "filename": "output.pdf",
    "record": {
      "field1": "value1",
      "field2": "value2",
      "checkbox1": true
    }
  }
]

Alternate Request Format

{
  "fileUrl": "https://example.com/path/to/template.pdf",
  "filename": "output.pdf",
  "record": {
    "field1": "value1",
    "field2": "value2",
    "checkbox1": true
  }
}

Getting Started

To use this API, send a POST request to the appropriate endpoint with your PDF template URL and form data.

For more information, check out the documentation.
